### 🧮 标签分布

| 感官类型 | 总数   | 正向样本 | 负向样本 |
|----------|--------|----------|----------|
| 触觉     | 23,764 | 18,901   | 4,863    |
| 视觉     | 70,321 | 56,331   | 13,990   |
| 味觉     | 5,741  | 4,340    | 1,401    |
| 嗅觉     | 1,876  | 1,701    | 175      |
| 听觉     | 2,129  | 1,802    | 327      |

这种结构化的标注方式使 CUCP 数据集适用于 **多分类**、**多标签** 和 **二分类情感分析** 等任务。
